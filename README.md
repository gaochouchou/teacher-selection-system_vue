# 毕业设计导师学生互选平台_vue

## 项目描述
实现毕业生毕业设计双向选择。教师可登陆后可查看数据统计信息，包含已选学生名单，课程权值，指导学生和已选学生数量，并能设置相应权重以及参数，从Excel中批量导入学生，单独内定学生，修改自己可指导学生的数量。学生登陆后可查看是否登陆，输入成绩，后台判定是否符合入选范围，并有友好的人机交互体验。
